Related Projects

A non-exhaustive list of projects that supplement Doobie:

  • DoobieRoll - collection of utilities to make working with Doobie / SQL even easier.
  • TableColumns - Ensure fields in your SQL are consistently named and ordered.
  • Assembler - Assemble SQL query results into hierarchical domain models.
  • doobie-typesafe - type-safe table definitions for Doobie.
  • otel4s-doobie - Otel4s integration.
